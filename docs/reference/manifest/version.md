# <a name="version-element"></a>Elemento Version

Especifica a versão de seu Complemento do Office.

**Tipo de suplemento:** Conteúdo, Painel de tarefas, Email

## <a name="syntax"></a>Sintaxe

```XML
<Version>n .n .n .n</Version>
```
