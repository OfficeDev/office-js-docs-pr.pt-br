# <a name="targetdialects-element"></a>Elemento TargetDialects

Define o idioma regional ou os idiomas compatíveis com esse dicionário.

**Tipo de suplemento:** Painel de tarefas

## <a name="syntax"></a>Sintaxe

```XML
<TargetDialects>
   ...
</TargetDialects>
```
