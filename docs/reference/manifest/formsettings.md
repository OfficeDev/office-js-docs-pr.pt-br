# <a name="formsettings-element"></a>Elemento FormSettings

Especifica o local de origem e as configurações de controle para o seu suplemento de email.

**Tipo de suplemento:** Email

## <a name="syntax"></a>Sintaxe

```XML
<FormSettings>
   ...
</FormSettings>
```
