## <a name="update-the-app"></a><span data-ttu-id="bc64b-129">Atualizar o aplicativo</span><span class="sxs-lookup"><span data-stu-id="bc64b-129">Update the app</span></span>

1. <span data-ttu-id="bc64b-130">Em seu editor de código, abra a pasta **My Office Add-in** que o Yo Office criou na raiz do seu projeto do Vue.</span><span class="sxs-lookup"><span data-stu-id="bc64b-130">In your code editor, open the folder **My Office Add-in** that Yo Office created at the root of your Vue project.</span></span> <span data-ttu-id="bc64b-131">Nessa pasta, você verá o arquivo de manifesto que define as configurações para o suplemento: **manifest.xml**.</span><span class="sxs-lookup"><span data-stu-id="bc64b-131">In that folder, you'll see the manifest file that defines the settings for your add-in: **manifest.xml**.</span></span>

3. <span data-ttu-id="bc64b-133">Abra o arquivo **index.htm** (localizado na raiz do seu projeto do Vue), adicione a marca `<script>` imediatamente antes da marca `</head>` e salve o arquivo.</span><span class="sxs-lookup"><span data-stu-id="bc64b-133">Open the file **index.html** (located at the root of your Vue project), add the following `<script>` tag immediately before the `</head>` tag, and save the file.</span></span>


3. <span data-ttu-id="bc64b-134">Abra **src/main.js** e *remova* o seguinte bloco de código:</span><span class="sxs-lookup"><span data-stu-id="bc64b-134">Open **src/main.js** and *remove* the following block of code:</span></span>

    ```js
    new Vue({
        el: '#app',
        components: {App},
        template: '<App/>'
    })
    ```
    
    <span data-ttu-id="bc64b-135">Depois adicione o código seguinte no mesmo local e salve o arquivo.</span><span class="sxs-lookup"><span data-stu-id="bc64b-135">Then add the following code in that same location, and save the file.</span></span> 
                                                         
    ```js
    const Office = window.Office
    Office.initialize = () => {
      new Vue({
        el: '#app',
        components: {App},
        template: '<App/>'
      })
    }
    ```

4. <span data-ttu-id="bc64b-136">Abra **src/App.vue**, substitua o conteúdo do arquivo pelo código a seguir, adicione uma quebra de linha ao final do arquivo (ou seja, após a marca `</style>`) e salve o arquivo.</span><span class="sxs-lookup"><span data-stu-id="bc64b-136">Open **src/App.vue**, replace file contents with the following code, add a line break at the end of the file (i.e., after the `</style>` tag), and save the file.</span></span> 

    ```html
    <template>
    <div id="app">
        <div id="content">
        <div id="content-header">
            <div class="padding">
            <h1>Welcome</h1>
            </div>
        </div>
        <div id="content-main">
            <div class="padding">
            <p>Choose the button below to set the color of the selected range to green.</p>
            <br/>
            <h3>Try it out</h3>
            <button @click="onSetColor">Set color</button>
            </div>
        </div>
        </div>
    </div>
    </template>

    <script>
    export default {
      name: 'App',
      methods: {
        onSetColor () {
          window.Excel.run(async (context) => {
            const range = context.workbook.getSelectedRange()
            range.format.fill.color = 'green'
            await context.sync()
          })
        }
      }
    }
    </script>

    <style>
    #content-header {
        background: #2a8dd4;
        color: #fff;
        position: absolute;
        top: 0;
        left: 0;
        width: 100%;
        height: 80px;
        overflow: hidden;
    }

    #content-main {
        background: #fff;
        position: fixed;
        top: 80px;
        left: 0;
        right: 0;
        bottom: 0;
        overflow: auto;
    }

    .padding {
        padding: 15px;
    }
    </style>
    ```
